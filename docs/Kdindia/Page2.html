who is especially skilled in one catagory of writing software. A Computer Programmer figures out the process of designing,
writing, testing, debugging/troubleshooting and maintaining the source code of computer programs. This source code is written
in a programming language so the computer can 'understand' it. Many companies use Computer Programmers to set up their websites
